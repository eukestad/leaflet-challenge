# USGS Earthquake Visualization
Use Leaflet to visualize USGS earthquake data

## Files in the Repo

- Leaflet-Step-1 - Basic visualization of earthquakes on a satellite map
- Leaflet-Step-2 - Step 1 with tectonic plates
